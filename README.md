# GiliGili

基于Flutter的GiliGili视频播放客户端，支持Android、iOS、Windows、macOS、Linux和Web平台。

本项目是对PiliPlus的改进版本，增加了更多功能和优化。

## 功能特性

- 支持视频播放、弹幕显示
- 首页推荐、热门、动态、追番等功能
- 用户登录、个人中心、历史记录
- 支持收藏、点赞、投币等互动功能
- TV模式支持
- 多平台支持
- 视频超分辨率
- 合并弹幕
- 会员彩色弹幕
- Cookie登录
- 显示视频分段信息
- 调节字幕大小
- 直播弹幕
- SponsorBlock功能

## 优化改进

- 专栏界面优化
- 私信界面优化
- 收藏面板优化
- 画中画(PIP)支持
- 视频封面优化
- 回复界面优化
- 系统通知优化
- 评论显示优化
- 亮度调节优化
- 视频播放优化
